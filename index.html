CPC#

A new CPC 464 emulator in 2021?

CPC# is an Amstrad CPC 464 emulator created just 37 years after the original machine was launched. Functionality is currently a little limited but the basic 1984 out of the box experience of a CPC 464 is there and (mostly) working with more to come...

Currently supported:

  • The 464 system, with original ROMs
  • Cassette support via CDT files
  • AY sound output
  • A really ugly, unfinished debugger

Not yet supported:

  • DSK files
  • Joystick port input
  • Alternative ROMs
  • Any customisation
  • Other systems in the CPC range
  • Much more...

Please report any issues found on the Github issues page or by emailing nrandle@dolbz.com
